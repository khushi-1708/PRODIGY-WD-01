# PRODIGY-WD-01
 “Task 1 – Responsive Landing Page”

## 💻 Description:
This is **Task 1** of my internship at **Prodigy InfoTech** under the **Web Development** domain.

## 🔧 Technologies Used:

- CSS3
- Flexbox
- Media Queries

## 🧠 What I Learned:
